# Nuxt 3 專案模版

## 必要條件
- Node.js v22 以上
- PNPM v9 以上 

## 主要 packages
- Nuxt 3.15.x
  > 啟用 4.0 版本的 Nuxt 特性
- Vue 3.5.x
- TypeScript 5.7.x
- Nuxt UI 3.0.0-alpha.x
- vitest 2.1.x

## Linter & Formatter
- ESLint 9.17.x - 程式碼品質檢查

## PNPM 指令
```sh
# 安裝依賴
pnpm install

# 開發模式
pnpm dev

# 建置專案
pnpm build
```
